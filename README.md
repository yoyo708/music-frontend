# 🎵 Music Practice Tracker｜音樂練習打卡板

一個用 Vue.js 製作的音樂練習紀錄小工具，
專為音樂人設計，簡單紀錄每週的練習狀況，
畫面可愛、操作輕鬆，讓練習也變成一種療癒！

## 🧩 技術堆疊
- Vue 3
- Vite
- Axios
- Tailwind CSS

## 🌟 功能特色
- ✅ 每週練習天數勾選
- 📝 支援練習筆記記錄
- 📈 可串接後端儲存紀錄（MongoDB）
